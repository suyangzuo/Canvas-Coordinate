# Canvas 坐标系可视化交互式工具

一个交互式的Canvas坐标系可视化工具，用于展示和理解世界坐标系与局部坐标系的概念，以及它们在Canvas绘图中的应用。

## 功能特点

- **双坐标系展示**：同时显示世界坐标系和局部坐标系，直观展示两者之间的关系
- **交互式操作**：支持拖动、缩放和旋转矩形对象
- **实时坐标显示**：实时显示鼠标位置和矩形参数
- **视觉辅助**：通过网格、刻度和标记点辅助理解坐标系统

## 主要功能

### 坐标系可视化
- **世界坐标系**：以Canvas画布左上角为原点(0,0)，显示全局坐标系统
- **局部坐标系**：以矩形中心为原点，跟随矩形移动和旋转，展示局部坐标系统

### 交互操作
- **拖动矩形**：点击矩形内部并拖动可改变其位置
- **缩放矩形**：拖动矩形的角点或边缘可调整其大小
- **旋转矩形**：拖动矩形上方的旋转控制点可旋转矩形

### 显示控制
- **坐标信息开关**：控制是否显示坐标信息面板
- **坐标背景开关**：控制是否显示坐标信息背景

## 技术实现

- 使用HTML5 Canvas API进行图形绘制
- 采用面向对象的方式组织代码，提高可维护性
- 响应式设计，适配不同屏幕尺寸
- 高DPI屏幕支持，确保图形清晰度

## 使用方法

1. 直接在浏览器中打开index.html文件
2. 使用鼠标与矩形进行交互：
   - 拖动矩形内部可移动位置
   - 拖动角点或边缘可调整大小
   - 拖动上方控制点可旋转矩形
3. 通过右下角的控制面板切换显示选项
